// Welcome to the intersection of dream and reality, REM! //

Contained inside this repo are two interactive Max/MSP samplers I created while I was new to Max/MSP in order to generate more interesting creature sounds and ambiences for the various horror related projects I was working on. I'd also like to consider this project a visual art / sound art piece. The interfaces takes any sample provided by its user and transforms them into dreamlike soundscapes using filtering, visually and sonically designed to somewhat evoke an atomic age of digital surrealism.

## Requirements
- Built using Max version 9.0.8 (82d284efd00) (arm64 mac)
- Mac OS X Version 14.3.1 (Build 23D60) arm64

## How to run
1. Download or clone this repository
2. Open "RapidEyeMovement.maxpat" or "SleepTalk.maxpat"
3. Turn on ezdac and any other control found within the patcher
4. If you'd like to swap out your sample, click on the "replace" button on the [buffer~]
5. Create!

## Notes
This project depends on local image assets stored in each respective folder.
Please keep the folder structure intact.
